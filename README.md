# V4 - Fathom Calls Collector

## Overview
An automated system that collects all Fathom call recordings and transcripts, organizing them in Google Drive with notifications for personal business review and analysis.

## Problem
Lack of centralized, organized storage for call recordings and transcripts for business reflection and analysis

## Target User
Individual business professional seeking organized call storage and review system

## Features
- Automatic Fathom call sync to Google Drive with date-based folders
- Transcript extraction and text file storage
- Gmail notification system with call summaries and file links
- Basic call metadata tagging in file names
- Twilio SMS notifications when calls finish processing
